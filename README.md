# 🛵 Proyecto CityScooter - CESDE

Este proyecto es un sistema de gestión para la renta de scooters eléctricos, desarrollado como parte de la evaluación individual de software.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** Java 21 (Temurin)
* **Framework:** Spring Boot 4.0.3
* **Gestión de Dependencias:** Gradle / Maven
* **Arquitectura:** Modelo en capas (Dominio, Servicios, Vista, Utilidades)

## 🏗️ Estructura del Proyecto
* **Dominio:** Contiene los POJOs (Scooter, Vehicle, PickupPoint, User, etc.) siguiendo el diagrama UML.
* **Servicios:** Lógica de negocio para la gestión de flota y puntos de recogida.
* **Utilidades:** Clases para lectura de consola y formateo de fechas.

## 🛠️ Cómo ejecutar
1. Clonar el repositorio.
2. Ejecutar `ScooterCityMain.java` desde el IDE (IntelliJ/Eclipse).
